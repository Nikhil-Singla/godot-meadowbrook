# Meadowbrook

A dynamic flying adventure game developed with the Godot Engine, featuring seasonal transitions and atmospheric gameplay mechanics. Players navigate a pilot through cloud-filled skies using parachute controls.

## Overview

Meadowbrook is a vertical scrolling flight game where players must skillfully maneuver through dynamic cloud formations while managing altitude and avoiding obstacles. The game features a seasonal progression system that introduces visual variety throughout the experience.

## Key Features

- **Seasonal Progression**: Visual and atmospheric changes as players advance through different seasons
- **Health Management**: Strategic gameplay requiring careful navigation to preserve player health
- **Custom Physics**: Tailored movement mechanics for parachute behavior
- **Parallax Backgrounds**: Multi-layered scrolling system for enhanced depth perception
- **Sound Design**: Integrated audio feedback for player actions and environmental interactions

## Technical Implementation

**Game Engine**: Godot 3.x  
**Languages**: GDScript  
**Asset Creation**: Adobe Photoshop, Procreate, Canva

### Architecture Highlights

- Modular scene structure with reusable node components
- Global state management system for persistent game data
- Custom collision detection for borders and obstacles
- Event-driven sound effect system
- Scene-based menu navigation with transition effects

## Installation

Download and run the executable file from the [Exe](https://github.com/Nikhil-Singla/godot-meadowbrook/tree/main/Exe) folder.

- **Arrow Keys**: Directional movement control
